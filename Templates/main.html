What is AppyMD ?

First of all, AppyMD is a very small css kit with little javascript and jquery based on bootstrap 5 to give you a Mobile material design best on the new Android versions.
So if a new Android version comes out, except a new kit.

Remember the kit is best for mobile devices and tables, it won't work fine on websites except if you which to add more and more Bootstrap & CSS like I did.

Material Design for AppyMD.

Material design is unique since it creates 3D like widget on your screen. Now I think this is why
Tech giant google decided to push it forward by using it in all its Apps. Witness from google-playstore, google-drive etc yet thats not all, even in their websites and browsers. google chrome

Now google brings new styles for MD in each new android version. So AppyMD is to depend on Android version releases and bootstrap with alot of added CSS.
So this is kit Android 11.

How the kit helps.

AppyMD modifies and adds widgets that are Material Design components. They are nice looking on mobile devices with just a simple additional CSS.

Please promise me to have learnt CSS and HTML5 and also don't forget javascript so you can be a proud programmer and this kit is just to save you time since you will get very many projects coming in as time passes.

But the kit is very googd for making App. Try a simple one with a framework called CORDOVA that helps in turning it into an APK file and utilizing the API of the os.
